## General Information
The main purpose of the project is to practice hexagonal architecture, DDD, microservice paterns by implementing a banking system simulator. The idea is to create 2 microservices on java and kotlin and make the comunication between them using Kafka message brocker. 

Account project structure

- Spring-boot - version 2.7.5
- Lombock - version 2.0
- Kafka - version 3.0
